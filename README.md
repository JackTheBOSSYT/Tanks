<b>Info:</b> Tanks is a game where there are two tanks that two players are controlling. The goal is to destroy the other player<br>
<b>How To Play:</b>
<b>PLAYER 1</b>
<b> W = Forward</b>
<b> A = Left</b>
<b> S = Back</b>
<b> D = Right</b>
<b> SPACE = Shoot</b>
<b>    |      </b>
<b>PLAYER 2</b>
<b> ↑ = Forward</b>
<b> ← = Left</b>
<b> ↓ = Back</b>
<b> → = Right</b>
<b> ENTER = Shoot</b>
<b>    |      </b>
